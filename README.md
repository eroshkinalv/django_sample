# Проект django_sample

## Описание:

Проект django_sample - это учебный проект на Python, который представляет собой основу для сайта интернет-магазина c использованием фреймворка Django. 

- Созданы шаблон и контроллер для отображения домашней страницы.
- Созданы шаблон и контроллер для отображения страницы с контактной информацией.
- Для стилизации страниц использован Bootstrap.

## Установка:

1. Клонируйте репозиторий:
```
git clone https://github.com/eroshkinalv/django_sample.git
```
